# health_app

This is a web-based health application where users can sign up either as a **user** or a **doctor**.

The platform has a social media-like interface where people can share health tips or anything related to well-being. Users can like posts, comment on them, share their own posts, and also save posts for later.

One of the main features is **"SAARTHI"**, which is a chatbot integrated into the system. It is based on machine learning and helps users with basic health-related queries. It can suggest exercises and recommend fruits or vegetables depending on the user’s condition.

The chatbot also provides:

* a list of doctors to consult (with contact details)
* suggestions for vendors who sell the recommended food items

The project is mostly completed and works as a prototype. However, it still needs improvements and refinements in some areas.

Overall, the main functionality has been implemented, and the application is close to being fully functional.
